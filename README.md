> [!IMPORTANT]
> Это форк [runetfreedom/russia-v2ray-rules-dat](https://github.com/runetfreedom/russia-v2ray-rules-dat).
>
> Зачем он нужен:
>
> - В этом форке настроена автоматическая публикация собранных артефактов в ветку `build`, чтобы на них можно было ссылаться напрямую через `raw.githubusercontent.com`.
> - В частности, публикуются отдельные артефакты для `sing-box` (SRS), например: `https://raw.githubusercontent.com/resetand/russia-v2ray-rules-dat/build/sing-box/geosite-ru-blocked.srs`.
>
> Чем отличается от оригинального репозитория:
>
> - В оригинале основной способ распространения артефактов — релизы.
> - Артефакты `sing-box` там публикуются как архив, а не как отдельные raw-файлы для прямого подключения по URL.
